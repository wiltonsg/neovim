# Neovim IDE

NeoVim from scratch

Repository based settings: https://github.com/LunarVim/Neovim-from-scratch

### Depends on:

1. Neovim: https://neovim.io/
2. Node.js: https://nodejs.org/en/
3. powerline
4. xsel
5. fonts-inconsolata

### Install:
```
git clone https://github.com/wiltonsg/neovim.git ~/.config/nvim
```
```
nvim +PackerSync
```
